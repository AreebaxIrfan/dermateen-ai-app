
# Dermateen AI

AI-powered skin analysis platform that helps users detect acne severity, analyze skin conditions, and receive intelligent skincare insights using advanced AI models.

## Features

### AI Skin Analysis

* Real-time skin condition detection
* Acne severity assessment
* AI-generated skincare insights
* Multi-layer image analysis
* Smart recommendation engine

### Authentication & Security

* Secure user authentication
* Protected routes
* Session management
* User-specific analysis history

### Modern User Experience

* Responsive design
* Beautiful UI/UX
* Fast image uploads
* Real-time feedback
* Mobile-friendly experience

### Production Ready

* Next.js App Router
* TypeScript
* Supabase Integration
* Vercel AI SDK
* Middleware Protection
* Scalable Architecture
